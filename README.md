# Radhika Portfolio

A **graphic design portfolio** website showcasing the work of **Radhika Agrawal**.

## What's inside

- **Single-page portfolio** built with HTML, CSS, and JavaScript (Ethereal theme by HTML5 UP)
- **About** — intro, skills, and design approach
- **Education** — M.F.A. & B.F.A. from B.R. Ambedkar University
- **Experience** — 6+ years in education, media, and freelance design
- **Gallery** — sample work: social media posts, ads, flyers, and branding

## Skills & tools

Photoshop · Canva · CorelDRAW · Visual design · Branding

## How to view

1. Open `portfolio/index.html` in a browser, or  
2. Host the `portfolio` folder on any static hosting (e.g. GitHub Pages) and visit the site URL.
